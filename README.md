# obsidian-plugin

utools

## ✨ 特性

- 🌈 开箱即用的`Vite+Vue3`的`Utools`插件开发模板
- ⚡ 开发环境自动注入`HMR`字段
- 🦍 自动构建`upx`包
- 🧸 构建`utools/preload`文件，处理第三方依赖
- 🚀 构建桌面应用

## 🥩 开始使用

### 🔧 安装依赖

> 推荐使用`pnpm`包管理工具，如果您还没有安装可以执行`npm install -g pnpm`进行安装

```bash
pnpm install
```

### 🛫 启动项目

```bash
pnpm dev
```

### 📦 打包项目

```bash
pnpm build
```
